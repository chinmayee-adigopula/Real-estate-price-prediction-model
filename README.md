# Bengaluru House Price Prediction

A house-price prediction project for Bengaluru using several regression and ensemble models.

## Project overview
This repository contains:
- Jupyter notebooks with EDA and model experiments (`.ipynb`)
- Trained model files (Pickle `.pkl`) â€” consider removing or hosting large models elsewhere
- A sample dataset: `Bengaluru_House_Data.csv`
- An app script (`App.py`) to serve predictions
- Supporting files: animation JSON, images, and demo media

## Models included
- Linear Regression (with and without PCA)
- Ridge, Lasso (with and without PCA)
- Decision Tree, Random Forest, Gradient Boosting (with and without PCA)
